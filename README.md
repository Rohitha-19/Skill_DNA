# 🚀 Skill_DNA

**Smart Competency Diagnostic & Profile Score Calculator**

A modern AI-powered Recruitment Intelligence Platform that evaluates candidates using competency-based assessments and provides hiring readiness evaluation, role matching, rankings, analytics, and recommendations.

---

## ✨ Features

* 👤 Candidate Dashboard
* 🧑‍💼 Recruiter Dashboard
* 📊 Competency Score Calculation Engine
* 🎯 Hiring Readiness Analysis
* 🏆 Candidate Ranking & Categorization
* 📈 Interactive Analytics & Charts
* 🤖 AI-Powered Recommendations
* 🔒 Role-Based Access Control
* 📱 Fully Responsive SaaS UI

## 🚀 Getting Started

### Prerequisites

Make sure you have installed:

* Node.js (v18 or later)
* npm

### Installation

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open your browser and visit:

```text
http://localhost:5173
```
